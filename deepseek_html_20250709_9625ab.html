React Hooks 完全指南：从入门到精通

张明 2023年10月15日 8分钟阅读 24条评论

React Hooks 自 React 16.8 版本引入以来，彻底改变了我们编写 React 组件的方式。它们提供了一种更直接的方式来使用 React 的特性，如状态和生命周期方法，而无需编写类组件。本指南将带你从基础概念到高级技巧，全面掌握 React Hooks。

什么是 React Hooks？

React Hooks 是让你在函数组件中"钩入" React 状态及生命周期等特性的函数。它们使你在无需修改组件结构的情况下复用状态逻辑，解决了在组件之间复用状态逻辑很难的问题。

React Hooks 示意图

基础 Hooks

useState

useState 是最基本的 Hook，它允许你在函数组件中添加 React 状态。

import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>
        Click me
      </button>
    </div>
  );
}

useEffect

useEffect Hook 让你在函数组件中执行副作用操作（数据获取、订阅或者手动修改DOM等）。

import React, { useState, useEffect } from 'react';

function Example() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    document.title = `You clicked ${count} times`;
  });

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>
        Click me
      </button>
    </div>
  );
}

高级 Hooks

useContext

useContext 让你能够订阅 React 的 Context 而不需要使用嵌套的组件。

useReducer

useReducer 是 useState 的替代方案，适用于复杂的状态逻辑。

提示：使用 useReducer 管理包含多个子值的状态对象，或者下一个状态依赖于之前的状态时特别有用。

自定义 Hooks

自定义 Hook 是一个函数，其名称以 "use" 开头，函数内部可以调用其他的 Hook。通过自定义 Hook，可以将组件逻辑提取到可重用的函数中。

import { useState, useEffect } from 'react';

function useWindowWidth() {
  const [width, setWidth] = useState(window.innerWidth);

  useEffect(() => {
    const handleResize = () => setWidth(window.innerWidth);
    window.addEventListener('resize', handleResize);
    return () => {
      window.removeEventListener('resize', handleResize);
    };
  }, []);

  return width;
}

Hooks 最佳实践

  • 只在最顶层使用 Hook
  • 只在 React 函数中调用 Hook
  • 使用 linter 插件来强制执行 Hook 规则
  • 将复杂逻辑拆分为多个 effect
  • 使用 useCallback 和 useMemo 优化性能

通过本指南，你应该对 React Hooks 有了全面的了解。Hooks 不仅简化了 React 组件的编写，还提供了更强大的代码复用能力。随着 React 生态的发展，Hooks 已经成为现代 React 开发的基石。
